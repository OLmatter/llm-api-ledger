## 易混淆点

- **Z.AI ≠ 国内智谱开放平台（bigmodel.cn）** — 两套账号、两套额度、两套计费体系互不互通。限流数值（80/400/1600 5h；400/2000/8000 周）和档位倍率（Lite:Pro:Max = 1:5:20）完全一致，只是币种和线路不同。
- **海外版 Z.AI 对 Claude Code / 海外编码 Agent 比国内版更稳** — 国内智谱对海外客户端风控更严。
- **ZCode 客户端在 Z.AI 上同样可用** — 0.67 折算权益跟国内智谱完全一致（高峰 3x→2x，非高峰 1x→0.67x，等效额度 1.5 倍），榜单 Z.AI 套餐会显示 ZCode×1.5 子行。
- 邀请码：火山方舟、MiniMax 有共建邀请计划（见 LeaderBoard 邀请码列）；其余三家（智谱 / Kimi / Z.AI）官方不开放邀请码。
