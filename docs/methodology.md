## 三级限额

火山方舟等厂商使用三级独立限流：
- **5h 窗口**：5 小时滚动窗口
- **周窗口**：自然周或 7 天滚动（厂商不同）
- **月窗口**：自然月或 30 天滚动

三者独立判断，任一触顶即锁，不会累积到下一级。

## 次数 vs Tokens

| 维度 | 来源 | 可信度 |
|---|---|---|
| 次数限额 | 厂商官方公布 | 厂商可改定义，仅参考 |
| Token 总额度 | 探针实测 | 真实使用反推，可信 |

**不要用次数比例反推 token 比例**。厂商可能改「一次请求」的定义（比如一次 = 100 tokens），导致次数和 tokens 的比例关系不稳定。

## 可信度分级

| 等级 | 条件 | 用途 |
|---|---|---|
| **low** | 单用户单次反推 | 参考，不进排序硬依据 |
| **medium** | 3+ 独立 user_hash 聚合 | 可作排序参考 |
| **high** | GitHub PR 验证 + 持续 30 天达标 | 绿标认证门槛 |
