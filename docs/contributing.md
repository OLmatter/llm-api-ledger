## 三种贡献方式

1. **装探针跑数据** — 装 → 跑 7+ 天 → 导出 PR 包 → 提 PR
2. **加新套餐** — 在 `data/vendors/` 和 `data/plans/` 加 YAML
3. **改榜单前端** — `docs/` 目录的 VitePress 站点

## 铁律（不能违反）

- **token 永不上传**：只上传聚合指标 + token 后 4 位
- **邀请码不影响排序**：没邀请码的套餐照样上榜
- **次数 ≠ tokens**：厂商公布的次数限额仅供参考，token 总额度需独立实测
- **单点不可信**：单用户单次反推标 `credibility: low`，不进排序硬依据
